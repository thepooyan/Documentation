`ctrl + p`      find file
`ctrl + shift + p`       commands
`ctrl + g`   goto line

command pallet:
@ > symbols
[empty] > files
: > lines


`ctrl + shift + .`    navigate through symbols
`ctrl + p, @` or `ctrl + shift + O`              search through symbols
`ctrl + p, #`              global search through symbols


`ctrl + l`    select line by line
